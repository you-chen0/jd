# 禁毒连连看 — 重构方案

## 一、设计概念："案卷档案"

将整个游戏包装为**缉毒案件情报终端**，玩家以「案件调查员」身份进入，通过辨认毒品照片与名称来完成证据归档。

### 核心隐喻

| 游戏元素 | 设计隐喻 |
|----------|----------|
| 游戏整体 | 案件情报分析系统 |
| 卡片网格 | 证据陈列板 |
| 图片卡 | 物证照片 |
| 文字卡 | 物证标签 |
| 配对成功 | 证据确认归档 |
| 关卡通关 | 案件告破 |
| 进度条 | 案件调查进度 |

### 设计基调

**严肃、精密、高技术感** — 不是娱乐化游戏，而是一个让人认真对待禁毒教育的严肃工具。

## 二、视觉设计系统

### 2.1 色彩体系

```css
--bg-base:       #0a0b14;   /* 深邃底色 — 近乎黑，带微弱蓝调 */
--bg-surface:    #11121f;   /* 表面层 — 微提亮 */
--bg-card:       #181929;   /* 卡片底色 */
--border:        #252540;   /* 默认边框 */
--border-hover:  #c0392b;   /* 悬停/选中边框 — 执法红 */

--text-primary:  #e4e4ec;   /* 主文字 — 暖白 */
--text-secondary:#787890;   /* 次文字 — 冷灰 */
--text-muted:    #4a4a60;   /* 禁用文字 */

--accent-red:    #c0392b;   /* 执法红 — 危险/警告 */
--accent-gold:   #d4a017;   /* 琥珀金 — 高亮/选中 */
--accent-green:  #27ae60;   /* 深翠绿 — 成功/完成 */
--accent-blue:   #225588;   /* 情报蓝 — 信息/链接 */

--glow-red:      0 0 18px rgba(192, 57, 43, 0.5);
--glow-gold:     0 0 18px rgba(212, 160, 23, 0.5);
--glow-green:    0 0 14px rgba(39, 174, 96, 0.4);
```

### 2.2 字体系统

| 用途 | 字体 | 字重 | 说明 |
|------|------|------|------|
| 主标题 | Noto Serif SC | 700 | 衬线体，传递权威与正式感 |
| 副标题/UI | Noto Sans SC | 400/500/700 | 无衬线，保证屏幕可读性 |
| 计时数字 | JetBrains Mono | 500 | 等宽数字，技术/终端感 |

### 2.3 材质与氛围

**背景层：**
1. 底色：`#0a0b14` 纯色
2. 径向光晕：从中心发散微弱亮光（模拟屏幕辉光）
3. SVG 噪点纹理：3% 不透明度叠加（胶片颗粒感）
4. 证据板网格线：1px 线条，50px 间距，2% 不透明度

**玻璃态面板：**
- `background: rgba(255,255,255,0.03)`
- `backdrop-filter: blur(12px)`
- 细微的 `1px border`

**卡片：**
- 暗色底 + 极细边框
- 内置阴影产生深度
- 四角 L 形标记（物证照片风格）
- 悬停时边框切换到红色 + 微弱光晕

## 四、动画系统

### 4.1 页面加载动画

| 元素 | 动画 | 延迟 | 时长 |
|------|------|------|------|
| 标题 | 从上方 fade in + slide down | 0ms | 400ms |
| 副标题 | fade in | 150ms | 300ms |
| 案卷卡片 1 | fade up + scale 0.95→1 | 200ms | 350ms |
| 案卷卡片 2 | 同上 | 300ms | 350ms |
| 案卷卡片 3 | 同上 | 400ms | 350ms |
| 按钮 | fade in | 500ms | 300ms |

### 4.2 游戏卡片入场

每张卡片错帧 40ms，从下往上淡入（fade + translateY）。

```
卡片 0: delay 0ms
卡片 1: delay 40ms
卡片 2: delay 80ms
...
```

### 4.3 庆祝粒子系统

**火花粒子（30个）：**
- 从屏幕中心爆发
- CSS 动画：radial-out + gravity-drop
- 颜色：红、金、绿随机
- 大小：4-8px
- 持续：2s

**彩屑（50个）：**
- 从顶部下落
- 随机水平偏移 + 旋转
- 颜色：红、金、绿、蓝、白
- 大小：6-12px
- 持续：3s

## 五、交互细节

1. **卡片悬停光标**：`cursor: pointer`，图片卡片悬停时轻微放大图片内容
2. **按钮按压**：`transform: scale(0.96)` + 阴影收缩
3. **分数跳动**：得分变化时数字做微型弹跳动画
4. **连击升温**：连击 ≥ 3 时，combo 数字变大并脉冲
5. **计时器紧迫**：< 15 秒时数字变红 + 脉冲 + 屏幕边缘红晕
6. **关卡选择反馈**：点击锁定关卡 → 抖动 + 提示文字弹出
7. **自定义滚动条**：细条样式，匹配整体暗色主题

## 六、技术实现

### 文件结构
```
禁毒连连看.html    （单文件约 1000 行）
├── <style>         CSS (~450 行)
│   ├── 变量/重置
│   ├── 背景/纹理
│   ├── 进度指示器
│   ├── 案卷卡片
│   ├── 统计栏（玻璃态）
│   ├── 游戏网格/卡片
│   ├── 按钮
│   ├── 结果界面
│   ├── 弹窗
│   ├── 动画关键帧
│   └── 响应式
├── <body>          HTML (~150 行)
└── <script>        JS (~400 行)
    ├── 关卡数据
    ├── 游戏状态
    ├── localStorage 持久化
    ├── 卡片生成/渲染
    ├── 选择/匹配逻辑
    ├── 计时/计分
    ├── 结束判定
    ├── 庆祝粒子系统
    └── 初始化
```

### 兼容性
- 目标浏览器：Chrome 90+, Safari 15+, Firefox 90+
- `backdrop-filter` 需降级处理（Firefox 不支持时使用纯色背景）
- `SVG 噪点滤镜` 全平台兼容
- 触摸设备：所有 `:hover` 效果降级处理

### 性能
- 卡片动画使用 CSS transform（GPU 加速）
- 庆祝粒子用 JS 创建 DOM 元素 + CSS 动画（避免 canvas 复杂计算）
- 噪点纹理静态 SVG，无运行时开销
